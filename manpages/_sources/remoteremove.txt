
.. _geogig-remote-remove:

geogig-remote-remove documentation
##################################



SYNOPSIS
********
geogig remote rm <name>


DESCRIPTION
***********

Remove the remote named <name>. All remote-tracking branches and configuration settings for the remote are removed.
